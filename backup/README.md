# PartyPlay backups

## `partyplay-multiplayer/` (June 2026)

Full snapshot of the **original Kahoot-style multiplayer** version:

- Host + many players compete simultaneously
- Quiz, reaction, tap, and arcade modes
- Cumulative scoring and win targets

To run the old version locally:

```bash
cd backup/partyplay-multiplayer
npm install
npm start
```

The live app in the repo root was updated to **live showcase mode** (one player at a time, projected on the big screen).
